# General Web Search Strategy

## Query Templates
- `"{topic}" review 2026`
- `"{topic}" comparison alternatives`
- `"{topic}" pros cons`
- `"{topic}" tutorial getting started`
- `"best {topic}" OR "top {topic}"`

## Source Priority
1. Official documentation sites
2. GitHub repositories (README, issues, discussions)
3. Tech news sites (TechCrunch, InfoQ, The Register)
4. Developer blogs with author credentials
5. Community forums (Reddit, HN, Dev.to)

## Tips
- Add current year to queries for recent content
- Use quotes for exact phrase matching
- Try both the full name and common abbreviations
