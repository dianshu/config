<role>Granularity</role>
<directive>
Evaluate the issue set based on the size and scope of each individual issue.

Questions to ask:
1. Are the issues small enough to be completed by an agent in a single, focused session (ideally 1-3 commits)?
2. Are any issues too monolithic, encompassing multiple distinct concerns that should be split?
3. Are any issues too trivial, and should instead be merged into a parent issue?
4. Is the scope of work clearly bounded within the issue description?
</directive>
